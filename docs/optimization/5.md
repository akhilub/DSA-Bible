## Example 1

1. `splines=false` doesn't work well with labels - it often causes unexpected behavior
1. `splines=line` is better for straight edges with labels
1. The layout engine (dot, neato, fdp, etc.) significantly affects how edges are drawn
1. Increasing `nodesep` and `ranksep` gives more space for straight edges
1. Using `labelfloat=true` can help position labels better

```dot
graph G {
    rankdir=TB;
    node[shape=circle style="rounded,filled" color="lightgoldenrodyellow" ]
    splines="line";
    K=1;

    // Increase spacing between nodes
    nodesep=1.5;  // Horizontal spacing between nodes (default is 0.25)
    ranksep=1.5;  // Vertical spacing between ranks (default is 0.5)

    1[shape=circle color=palegreen1 label="1"]
    2[shape=circle color=palegreen1 label="2"]
    3[shape=circle color=palegreen1 label="3"]
    4[shape=circle color=palegreen1 label="4"]
    5[shape=circle color=palegreen1 label="5"]
    6[shape=circle color=palegreen1 label="6"]
    7[shape=circle color=palegreen1 label="7"]
    8[shape=circle color=palegreen1 label="8"]

    {rank=same; 1, 2, 3 }
    {rank=same; 5, 6, 7 }

    //GraphViz dont support labelling edges in compact form

    1 -- {2 [len=2.0] 4 [len=2.0] 5 [len=2.0]};
    2 -- {3 [len=2.0] 4 [len=2.0] 6 [len=2.0]};
    3 -- {6 [len=2.0] 7 [len=2.0]};
    4 -- {5 [len=2.0] 6 [len=2.0]};
    5 -- {6 [len=2.0] 8 [len=2.0]};
    6 -- {7 [len=2.0] 8 [len=2.0]};
    7 -- 8 [label="25" len=2.0];
}
```

- `nodesep` for horizontal spacing
- `ranksep` for more vertical spacing
- `len` values for longer individual edges

## Example 2

```dot
graph G {
    rankdir=TB;
    node[shape=circle style="rounded,filled" color="lightgoldenrodyellow"];
    splines=false;
    K=1;

    nodesep=0.5;
    ranksep=0.5;

    1 -- 2 [label="30" len=2.0 labeldistance=1.5 labelangle=0];
    1 -- 4 [label="24" len=2.0 labeldistance=1.5 labelangle=0];
    1 -- 5 [label="25" len=2.0 labeldistance=1.5 labelangle=0];
    2 -- 3 [label="27" len=2.0 labeldistance=1.5 labelangle=0];
    2 -- 4 [label="28" len=2.0 labeldistance=1.5 labelangle=0];
    2 -- 6 [label="27" len=2.0 labeldistance=1.5 labelangle=0];
    3 -- 6 [label="26" len=2.0 labeldistance=1.5 labelangle=0];
    3 -- 7 [label="30" len=2.0 labeldistance=1.5 labelangle=0];
    4 -- 5 [label="29" len=2.0 labeldistance=1.5 labelangle=0];
    4 -- 6 [label="21" len=2.0 labeldistance=1.5 labelangle=0];
    5 -- 6 [label="30" len=2.0 labeldistance=1.5 labelangle=0];
    5 -- 8 [label="30" len=2.0 labeldistance=1.5 labelangle=0];
    6 -- 7 [label="28" len=2.0 labeldistance=1.5 labelangle=0];
    6 -- 8 [label="36" len=2.0 labeldistance=1.5 labelangle=0];
    7 -- 8 [label="25" len=2.0 labeldistance=1.5 labelangle=0];

    1[shape=circle color=palegreen1]
    2[shape=circle color=palegreen1]
    3[shape=circle color=palegreen1]
    4[shape=circle color=palegreen1]
    5[shape=circle color=palegreen1]
    6[shape=circle color=palegreen1]
    7[shape=circle color=palegreen1]
    8[shape=circle color=palegreen1]

    {rank=same; 1, 2, 3 }
    {rank=same; 5, 6, 7 }
}

```
